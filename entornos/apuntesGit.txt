PARA PONER ALIAS A UN COMMIT

$ git config --global alias.c commit
le ponemos el alias de c a commit

luego lo usariamos poniendo 
git c -m"ejemplo alias"

para deshacerlo utilizamos 
git config --global --unset alias.c
